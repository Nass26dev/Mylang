# Documentation of my programming language
This project consists of creating my own programming language in French which is my native language
## Types of variables
-`nb` : Integer.  
-`chaine` : String.  
-`char` : char.  
-`bool` : boolean.  

## Loop
- **While loop** :
    ```mylang
    tant que (i != 12) {
        // instructions
    }
- **For loop** :
    ```mylang
    pour (i = 0, i <= 20, i++) {
        // instructions
    }
## Terms
- **If** :
    ```mylang
    si (i < 10) {
        // instructions
    }
- **Elif/Else if** :
    ```mylang
    sinon si (i == 10) {
        // instructions
    }
- **Else** :
    ```mylang
    sinon {
        // instructions
    }
