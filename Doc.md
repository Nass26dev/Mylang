# Documentation of my programming language

## Types of variables
-`nb` : Integer.  
-`chaine` : String.  
-`char` : char.  
-`bool` : boolean.  

## loop
- **while loop** :
    ```mylang
    tant que (i != 12) {
        // instructions
    }
- **for loop** :
    ```mylang
    pour (i = 0, i <= 20, i++) {
        // instructions
    }

## terms
- **if** :
    ```mylang
    si (i < 10) {
        // instructions
    }
- **elif/else if** :
    ```mylang
    sinon si (i == 10) {
        // instructions
    }
- **else** :
    ```mylang
    sinon {
        // instructions
    }
